## 这是什么

这是一个 **Claude Code 自定义 Skill**，解决的是药品社媒投放中「从策略到内容落地」的最后一公里问题：

- 你已经有了投放策略报告（明确了利益点、人群、合规红线）
- 你已经有了达人明细表（筛好了要合作的达人）
- 你需要的是：为每位达人写出 **符合其人设、覆盖策略利益点、彼此不雷同、合规可发布** 的笔记内容

这个 Skill 把这件事从头到尾自动化了。

---

## 为什么需要它

| 痛点 | 本 Skill 的解决方式 |
|------|-------------------|
| 50个达人要写50条不同的笔记，人工写太慢 | 批量生成，每条根据达人画像定制 |
| 写完了不确定有没有踩合规红线 | 七维度自动审核 + 自动修正 |
| 多人协作容易出现内容雷同 | 四层去重机制（标题/开头/结构/话题） |
| 药品合规规则复杂，人工容易遗漏 | 内置完整法规知识库 + 策略专项红线 |
| 不同达人的人设和口吻差异大 | 9种人设风格自动匹配 |
| 每换一个药品就要重新理解规则 | 完全通用模板，所有信息从策略报告读取 |

## 快速开始

在 Claude Code 中输入：

```
/xhs {达人明细表路径} {投放策略报告路径}
```

**示例：**

```
/xhs D:\项目\Q3投放\达人明细.xlsx D:\项目\Q3投放\某药品_投放策略报告.md
```

然后等待即可。Skill 会自动完成全部流程，最终输出更新后的表格和执行汇总。

```markdown
## 五、总结
```

**Skill 从策略报告中提取什么：**

| 提取内容 | 来源位置 | 在 Skill 中怎么用 |
|---------|---------|-----------------|
| 药品名称、通用名 | 一、基础档案 | 构建高风险关键词清单 |

### 各步骤详解

#### Step 1：解析达人画像

从表格每一行提取关键信息，构建结构化画像：

```
达人画像 = {
  序号, 昵称, 平台, 科室, 账号简介,
  地区, 粉丝量, 类型标签, 达人层级
}
```

#### Step 2：分配内容角度

这是"不雷同"的核心保障。角度池**动态生成**——从策略报告的利益点、内容方向、药品特性中构建：

- 每个策略利益点至少对应 1 个内容角度
- 每个内容方向至少对应 1 个角度
- 根据药品特性补充通用角度（机制科普、副作用管理、误区澄清等）
- 角度池数量 ≥ 达人数（不够时自动扩展：细分、组合、平台差异化、场景变体）

**分配规则：**
- 同科室达人不重复角度
- 头部达人 → 深度专业内容；腰部 → 实用易懂
- 匹配策略报告的达人矩阵角色定位
- 重复达人只生成一条

#### Step 3：按人设生成内容

根据达人特征自动匹配 9 种风格之一：

| 达人特征 | 内容风格 | 品牌处理 |
|----------|----------|----------|
| 头部+主任/教授 | 权威科普，引用指南数据 | 只提品类通用名 |
| 头部+硕博 | 学术科普，机制解析 | 只提品类 |
| 腰部+主治医师 | 实用指导，临床经验 | 模糊化泛称 |
| 腰部+泛科普 | 生活化科普，场景切入 | 绝不提品牌 |
| 非医KOC/博主 | 场景种草，生活经验 | 不提品牌 |
| 患者KOL/真实用户 | 第一人称经历分享 | 不提品牌 |
| 品牌号/官媒 | 正式科普，品牌定调 | 可提品牌不做疗效承诺 |
| 中医背景 | 中西结合视角 | 只提品类 |
| 跨科室医生 | 跨科关联科普 | 不提品牌 |

**生成时的硬性规则：**
1. 正文表述必须基于策略报告中的"用户语言"，不自行编造
2. 策略报告的 ❌ 表述绝对不出现
3. 每篇至少关联 1-2 个核心利益点
4. 禁止贬损竞品
5. 严格在适应症范围内，不暗示非适应症功效

#### Step 4：去重自检

生成全部内容后，四层交叉检查：

1. **标题去重** — 任意两篇标题不得有超过 3 个相同关键词
2. **开头去重** — 每篇使用不同切入方式（提问/数据/场景/案例/误区/对话/热搜）
3. **结构去重** — 混合使用不同体裁（清单体/故事体/问答体/科普体/对比体/时间线体/红黑榜体）
4. **话题去重** — 每篇的核心话题点不重复

发现雷同则自动调整角度重新生成。

#### 平台差异处理

| 维度 | 小红书 | 抖音 |
|------|--------|------|
| 格式 | 图文笔记，分段 + emoji | 短视频口播脚本 |
| 字数 | 300-800 字 | 200-400 字 |

## 合规审核详解

审核共覆盖 **7 个维度**，其中维度 1-5 为合规硬性检查（影响评分），维度 6 为必备要素（缺失即扣分），维度 7 为内容质量（辅助参考不扣分）。

### 维度 1：法律合规

依据《广告法》《药品管理法》《互联网广告管理办法》逐项比对：

- 品牌名 + 疗效 + 购买引导三者共存 → **高危**
- 疗效承诺/安全性断言 → **高危**
- 绝对化用语 → **高危**
- 虚构数据/未验证功效 → **高危**
- 诱导不当用药 → **高危**
- 伪装科普实为广告 → **高危**

### 维度 2：平台社区规范

- 站外引流/交易引导 → **高危**
- 低俗/擦边内容 → **高危**
- 未标注广告/合作 → **中**
- 标题党/敏感词 → **中**

### 维度 3：品牌露出合规

- 品牌名出现次数 ≤ 1 次，且不与疗效/购买同段
- 禁止药品包装特写、价格信息、促销暗示
- 禁止贬低竞品

### 维度 4：疗效表述

- 禁止量化承诺（"瘦X斤""X天见效"）
- 禁止治愈性承诺（"根治""痊愈"）
- 禁止安全性绝对化（"零副作用"）
- 禁止适用性断言（"所有人都适合"）

### 维度 5：高风险关键词扫描

关键词清单**动态构建**，包括：
- 本品品牌名 + 竞品品牌名（从策略报告提取）
- 通用违规词库（疗效承诺/时间承诺/绝对化/购买引导）
- 策略专项禁区词（从策略报告合规提醒的 ❌ 表述中提取）

### 维度 6：必备要素检查

- "遵医嘱"/"请在医生指导下使用" — 必须存在
- 正文字数 — 小红书 300-800 / 抖音 200-400
- 标签与内容相关性
- 个体差异声明（建议有）

### 维度 7：内容质量（不扣合规分）

- 开头吸引力、互动引导、结构清晰度
- 是否偏离分配角度

## 评分与风险判定

### 评分标准

| 分数区间 | 含义 | 对应结论 |
|---------|------|----------|
| 9-10 | 完全合规，内容质量好 | ✅ 通过 |
| 7-8 | 基本合规，有小瑕疵可优化 | ✅ 通过（附建议） |
| 5-6 | 存在需修改的问题 | ⚠️ 需修改 |
| 3-4 | 多处违规，需大幅修改 | ⚠️ 需修改 |
| 1-2 | 严重违规，建议重写 | ❌ 不建议发布 |

### 扣分规则

| 问题类型 | 扣分幅度 |
|---------|---------|
| 高严重程度问题（法律红线） | 每项 -2~3 分 |
| 中严重程度问题（平台规范） | 每项 -1~2 分 |
| 低严重程度问题（质量建议） | 每项 -0.5~1 分 |
| 缺失"遵医嘱"声明 | 直接 -2 分 |
| 命中高风险关键词且构成违规 | 每个 -1~2 分 |

### 风险等级

| 等级 | 含义 |
|------|------|
| 高 | 违法或平台红线，必须改，不然封号 |
| 中 | 社区规范问题，建议改，不然限流 |
| 低 | 质量瑕疵，改了效果更好 |

## 设计原则

| 原则 | 说明 |
|------|------|
| **完全通用** | 不内置任何药品信息，所有具体内容从策略报告动态读取，适用于任何药品 |
| **格式固定、内容灵活** | 两个输入文件的格式结构一致，内容随药品/项目自由变化 |
| **宁严勿松** | 合规审核采用从严标准，宁可误判也不漏判 |
| **双重保险** | 生成阶段内置合规约束 + 审核阶段全量复核，两道关卡 |
| **可追溯** | 每条修改都标注"改了什么、为什么改"，方便人工复核 |
| **不雷同** | 角度分配 + 四层去重，确保50条内容各不相同 |

## 常见问题

**Q：换一个药品需要改 Skill 本身吗？**
> 不需要。只需提供新药品的投放策略报告（格式相同，内容不同），Skill 会自动适配。

**Q：达人表格的列顺序能变吗？**
> 不能。Skill 按固定的 22 列顺序读取。如果你的表格列顺序不同，需要先调整为标准格式。

**Q：达人数量有上限吗？**
> 没有硬性上限。超过 10 人时会询问批量策略（全量/分批/指定范围），每完成 10 条输出进度。

**Q：审核不通过的内容怎么处理？**
> 自动生成修正版写入第 29 列。原始版本保留在第 25 列，方便对比。

**Q：生成的内容能直接发布吗？**

## 注意事项

1. **辅助工具定位** — 最终发布前建议人工确认，尤其是处方药相关内容
2. **合规标准时效性** — 法规和平台规则会更新，需定期校对 Skill 中的知识库是否过时

4. **三要素红线** — 品牌名 + 疗效 + 购买引导，三者同时出现 = 违法广告，任何时候不可触碰
5. **策略报告质量决定输出质量** — 策略报告中的利益点越清晰、用户语言越具体，生成的笔记越贴合需求
